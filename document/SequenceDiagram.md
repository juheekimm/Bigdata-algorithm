# Sequence Diagram

### 상품 검색 기능의 통신 흐름

```sequence
사용자 ->> Front : 1.사용자의 검색 요청
Front ->> Back : 2.axios를 사용하여 검색조건 및 검색어 전달
Back ->> DataBase : 3.검색 조건에 일치하는 데이터 조회
DataBase -->> Back : 4.검색 데이터 반환
Back -->> Front : 5.검색 데이터를 사용자가 지정한 방식(ex:json)으로 반환
Front -->> 사용자 : 6.검색 결과 반환

```



### 메세지 종류

- 동기 메세지 : 색칠된 화살표 (-▶)

- 비동기 메세지 : 빈 화살표 (->)

- 반환 메세지 : 점선으로 표기
